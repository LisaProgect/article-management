## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Running prisma

```bash
$ npx prisma generate

# development
$ yarn run db:migrate:dev
```

## Running seed

```bash
$ yarn run seed
```

## Project structure

### Modules

```

-- modules
---- dto
---- example-module
-------- controllers
-------- services
-------- types

```

Modules is the core of application. Each feature should be created as a separate module (Example: `modules/users` or `modules/auth`).

### Prisma

```
-- prisma
---- schema
---- seed
```
